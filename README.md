# FitPocket Streamlit

Budget-friendly, balanced 7-ELEVEN meal plans with BMR/TDEE calculator.

## Quick Start
1) pip install -r requirements.txt
2) streamlit run final_project/fitpocket_app_version1.py

## Data
- 711_food_data.csv (used by the app)

## Deploy
- Streamlit Community Cloud: set main file to final_project/fitpocket_app_version1.py
