# FitPocket Streamlit

Budget-friendly, balanced 7-ELEVEN meal plans with BMR/TDEE calculator. Earth-tone UI, quick plan generation, and QR-friendly sharing.

## Features
- BMR/TDEE 計算與熱量級距建議
- 依預算（低/中/高）生成最多三套一日三餐菜單
- 食物六大類涵蓋提示、熱量/價格明細
- 7-ELEVEN 商品資料集 (`711_food_data.csv`) 內建
- Earth tone UI + 插畫式 hero，適合報告展示或分享 QR

## Quick Start (local)

pip install -r requirements.txt
